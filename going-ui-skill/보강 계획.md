## 2. GoTrendGraph Start/SetData 실시간 패턴

**현재 상태**: JSON 구조만 존재. 실시간 스트리밍 C# 코드 없음.

**추가할 내용**:
- GoLineGraphSeries 구조 (Name, Alias, Color, Visible, Minimum, Maximum)
- Start<T>(template) → SetData<T>(data) → Stop() 전체 흐름
- OnUpdate에서 통신값 연결하는 실전 패턴
- Pause/Resume 제어

**핵심 포인트**:
- Start<T>의 T는 Series.Name과 매칭되는 프로퍼티를 가진 객체
- Interval(ms) 주기로 자동 수집
- SetData로 소스 객체 갱신 → 다음 Interval에 반영

## 4. GoLineGraph / GoTimeGraph 샘플

**현재 상태**: JSON 구조만 존재. GoLineGraphSeries 구조 미기재.

**추가할 내용**:
- GoLineGraphSeries 구조 (Name, Alias, Color, Visible, **Minimum, Maximum**)
- GoLineGraph: SetDataSource<T>(XAxisName, values) — XAxisName은 string
- GoTimeGraph: SetDataSource<T>(XAxisName, values) — XAxisName은 **DateTime**
- Series별 개별 Y축 범위 설정

**핵심 포인트**:
- GoLineGraphSeries는 GoGraphSeries와 달리 Minimum/Maximum 보유 (개별 Y축 스케일)
- GoTimeGraph의 XScale, XAxisGraduationTime은 TimeSpan 형식

## 5. GoGraphSeries / GoLineGraphSeries 구조 정의

**현재 상태**: 이름만 언급, 프로퍼티 정의 없음.

**추가할 내용** (1~4번 샘플에 자연스럽게 포함):

```
GoGraphSeries:
  - Name: string         (데이터 매칭 키)
  - Alias: string        (범례 표시명)
  - Color: string        (시리즈 색상)
  - Visible: bool        (표시 여부)

GoLineGraphSeries (GoGraphSeries 상속):
  - Minimum: double      (Y축 최소)
  - Maximum: double      (Y축 최대)
```
